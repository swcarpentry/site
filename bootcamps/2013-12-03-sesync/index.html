Where: SESYNC, Annapolis, MD

When: December 3-6. We will start at 9:00 and end at 4:30 each day.

Registration for this boot camp will be restricted to personnel from SESYNC and from the University of Maryland and USM.

{% include bootcamps/instructors.html %}

What: The SESYNC bootcamp will spend two days covering basic scientific computing topics including using command line/shell for automating tasks and redirecting output; versioning for tracking provenance and collaborating with others on code development; automated testing to ensure that code is functioning as expected; basics of python programming which can accelerate tasks in many software packages (e.g., ArcGIS, SQL, R); and potentially one or two other topics TBD. Potential additional topics include best practices in spreadsheet management, transitioning spreadsheets to Excel, SQL databases, and regular expressions for searching and cleaning data. The third and fourth days will be an open consultancy where participants would bring their projects, apply their new skills, and get on-the-spot help as needed.
