Admin is the staff at Software Carpentry's (virtual) headquarters who support boot camp hosts by finding instructors, setting up registration, and other tasks.

Preliminaries

Before the boot camp host has committed to holding a boot camp, Software Carpentry admin helps them decide when and where the boot camp should be, advises on likely cost, and provides other guidance.

Initial Setup

  • Find instructors. This should be done before the boot camp is publicly announced.
  • Find helpers, if the host doesn't think she can find any locally.
  • Set up the boot camp web page and check that the Github web page is correct.
  • Set up an Eventbrite event if required.
  • Post to the Software Carpentry announcements Twitter feed and the announcements mailing list.
  • Confirm that the host has agreed to the administration fee ({{site.admin_overhead}}, negotiable).

Before the Boot Camp

  • Set up a mailing list for the host, instructors, and helpers.
  • Make sure instructors know how to claim their travel expenses (if any).
  • Set up the pre-assessment questionnaire and ask the host to distribute it to learners.

A Week Before

  • If there are cancellations, offer tickets to people on the waiting list
  • Make sure that everyone's travel plans have been made, that the schedule is firm, that helpers are available, etc.

During

  • Check with the host and instructors by email on the first day to make sure everything is going as planned.
  • Remind the instructors to distribute the post-assessment questionnaire.

After

  • Get the names and email addresses of attendees and helpers and add them to our database. (If names and email addresses are unavailable, get the number of attendees and add that.)
  • Collect post-assessment questionnaire responses.
  • Make sure the instructors have filled in their post-assessment questionnaire, and record their responses.
  • Invite attendees to join the announcements mailing list.
  • Invite helpers to take part in instructor training.
  • Get a blog post from the host, an instructor, or another member of the boot camp team.
